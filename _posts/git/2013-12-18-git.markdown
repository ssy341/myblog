git clone git@github.com:ssy341/myblog.git  克隆项目到本地（项目名称即为文件夹名称）

git remote add origin git@github.com:你的github用户名/你的github项目名.git 

git branch 查看当前分支

git pull origin master(分支名) 

git add * 加入所有文件到缓存区

git commit -m "注释" 

git push -u origin master(分支名称)
